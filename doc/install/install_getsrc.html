Obtaining the Toast++ sources

The Toast++ SVN repository is located at http://sourceforge.net/p/toastpp/code/HEAD/tree/. To check out the head revision of the main branch, make sure you have an SVN client installed, and check out Toast++ into subdirectory "toastpp" with

svn checkout svn://svn.code.sf.net/p/toastpp/code/toast/trunk toastpp

The Sourceforge site also contains pre-compiled binaries of the latest stable release for Linux, Windows and MacOS. See the Toast++ Sourceforge site at http://sourceforge.net/projects/toastpp/ for details.

Additional information, documentation and tutorials can be found at the Toast++ home page.
