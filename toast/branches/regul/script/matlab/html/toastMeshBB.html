toastMeshBB

Return the extents of the bounding box of the mesh.

Syntax

[bbmin bbmax] = toastMeshBB (hMesh)

Parameters

[In] hMeshhandlemesh handle
[Out] bbminreal [d]Lower left corner of the bounding box.
bbmaxreal [d]Upper right corner of the bounding box.
d = domain dimension (2 or 3)

Notes

This function returns the minimum and maximum corners of the bounding box enclosing the mesh. The values of the bounding box corners are defined by the minimum and maximum mesh vertex coordinate values in each axis.

Note that for meshes containing isoparametric elements (i.e. elements with curved boundaries) the mesh bounding box may not be correctly represented by node positions. This is not taken into account.
