toastLineSearch

1-D approximate minimisation of an objective function Ψ(x) along a given search direction.

Syntax

[s pmin] = toastLineSearch (x0, d, s0, p0, func, ...)

Parameters

[In] x0real [n]current parameter estimate x0
dreal [n]search direction
s0realinitial step size
p0realinitial value of objective function Ψ(x0)
funcfunction pointercallback function for objective evaluation. Format:
function p = func(x, ...)
...any additional parameters to be passed to func
[Out] srealfinal step size
pminrealfinal value of objective function, Ψ(x0 + sd)

Notes

Performs an approximate minimisation of a function Ψ(x0 + s*d), where x0 is the initial parameter set, d is the search direction, and s is the step size being optimised.

x0 and d must be provided as vectors of equal size. The actual size n is of no importance to toastLineSearch, because it simply passes the trial step x0+s*d on to the callback function func.

toastLineSearch is independent of the actual realisation of Ψ. The calling function must provide a pointer to a function which evaluates Ψ for given x.

toastLineSearch works by first bracketing the minimum, and then performing a quadratic interpolation step. d is required to be a downhill direction in the vicinity of x0.
