toastMapSolToBasis

Map an image between two different basis representations: from s-basis to b-basis (see Basis definitions).

Syntax

bimg = toastMapSolToBasis (hBasis, simg)

Parameters

[In] hBasishandlebasis mapper handle
simgreal or complex [s]image (basis coefficients) in s-basis expansion
[Out] bimgreal or complex [b]image (basis coefficients) mapped into b-basis expansion
Dimensions: s = size of s-basis, b = size of b-basis.

Notes

This function maps from the (sparse) solution representation (excluding exterior voxels not overlapping the problem domain) to the (dense) regular grid (including exterior voxels). Exterior voxels are set to a value of 0.

Images can be real or complex-valued.

Because the only difference between s-basis and b-basis is the exclusion or inclusion of exterior voxels, the same functionality can be obtained by using the permutation vector returned by toastSolutionMask:

smask = toastSolutionMask(hBasis);
bimg(smask) = simg;

If the hBasis handle was created by a call to toastSetBasis with only a single regular grid parameter, then the g-basis and b-basis expansions are identical, and toastMapSolToBasis is equivalent to toastMapSolToGrid.
