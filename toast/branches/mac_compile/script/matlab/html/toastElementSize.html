TOAST > Functions > toastElementSize

toastElementSize

Return an array with mesh element sizes.

Syntax

s = toastElementSize(hMesh)

Parameters

[In] hMeshhandlemesh handle
[Out] sreal vectorarray of element sizes

Notes

Returns the areas (2-D meshes) or volumes (3-D meshes) of each element of a TOAST mesh in a column vector. Note that negative element sizes usually indicate an incorrect node order.
