toastRecon

The TOAST inverse solver for image reconstruction in optical tomography.

Syntax

toastRecon (prm)

Parameters

[In] prmstructcontains options and parameters for the inverse solver

Notes

This is a high-level function that contains the TOAST nonlinear iterative inverse solver for reconstructing absorption and scattering images from frequency-domain boundary data of light transillumination.

The solver uses a regularised Gauss-Newton scheme or nonlinear conjugate gradient scheme to iteratively minimise an objective function defined as a least-squares difference between measurements and forward model parameters for a given estimate of optical coefficients.

All model parameters and options are provided by the 'prm' input argument. For a description of the prm structure, see Parameter structure.

prm can either be generated manually, read from a file with toastReadParam, or defined interactively with a GUI tool, recon_gui.
