toastElementData

Return geometry data of a mesh element.

Syntax

[vtx,idx] = toastElementData(hMesh,el)

Parameters

[In] hMeshhandlemesh handle
elintegerelement index (≥ 1)
[Out] vtxreal matrixn x d matrix containing n d-dimensional node coordinates
idxinteger matrixface vertex index list

Notes

Returns the node coordinates, and the element vertex index lists for each face of a mesh element.

The returned vertex list is a real matrix of dimension n x d, where n is the number of nodes in the element, and d is the geometric dimension of the element (2 or 3).

The returned index list is an integer matrix of dimension m x s, where m is the number of element faces, and s is the maximum number of nodes associated with a face. The index list is 1-based. For faces containing fewer than s nodes, the unused entries are set to 0.
