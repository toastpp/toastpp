toastSetBasis

Define a mapper between FEM and reconstruction bases.

Syntax

hBasis = toastSetBasis(hMesh,bdim)
hBasis = toastSetBasis(hMesh,bdim,gdim)

Parameters

[In] hMeshhandlemesh handle
bdiminteger vector [2] or [3]raster dimensions of the b-basis grid
gdiminteger vector [2] or [3]raster dimensions of the g-basis grid
[Out] hBasishandlebasis mapper handle

Notes

A basis mapper provides an interface to map a scalar field, defined over the domain, from one basis expansion to another.

TOAST uses different basis definitions for the forward and inverse problem. The forward solution is computed on a finite element grid which may be unstructured (h-basis). The inverse solution is normally calculated on an independent regular grid (b-basis). Optionally, an additional high-resolution regular grid (g-basis) is used to improve the mapping between h and b-basis.

toastSetBasis creates a mapper instance. Parameter hMesh is a mesh handle which defines the h-basis. The inverse basis is a regular bi- or tri-linear basis of grid size bdim(1) x bdim(2) or bdim(1) x bdim(2) x bdim(3). The area of the b-basis is set to the bounding box of the mesh.

If the gdim parameter is provided, it is used to define a high-resolution regular grid (g-basis). gdim should be of higher resolution than bdim (typically an integer multiple of bdim in each dimension). If gdim is not specified, no intermediate grid is used for the mapping between h-basis and b-basis. The g-basis is usually only required if the b-basis has a low resolution, to avoid losses during the mapping process.
