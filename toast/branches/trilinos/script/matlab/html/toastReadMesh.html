toastReadMesh

Read a TOAST mesh from file.

Syntax

hMesh = toastReadMesh(meshfile)

Parameters

[In] meshfilestringmesh file name
[Out] hMeshhandlemesh handle

Notes

Reads a TOAST mesh from a file and returns a handle. The mesh handle cannot be used by MATLAB directly but can be passed to other TOAST functions.

After use, the mesh should be de-allocated with toastDeleteMesh.
