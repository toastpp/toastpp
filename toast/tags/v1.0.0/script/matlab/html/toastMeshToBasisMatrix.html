toastMeshToBasisMatrix

Returns the linear transformation matrix that maps a nodal image into a regular grid.

Syntax

M = toastMeshToBasisMatrix (hBasis)

Parameters

[In] hBasishandlebasis mapper handle
[Out] Mreal, sparse [b x h]transformation matrix
Dimensions: h = size of h-basis, b = size of b-basis.

Notes

Using the matrix returned by toastMeshToBasisMatrix for a basis transformation is equivalent to applying the toastMapMeshToBasis function.

Therefore, the operation

M = toastMeshToBasisMatrix (hBasis);
bimg = M * himg;
has the same result as
bimg = toastMapMeshToBasis (hBasis, himg)
(where himg is an image in nodal basis representation, and hBasis is a valid basis mapper handle).
