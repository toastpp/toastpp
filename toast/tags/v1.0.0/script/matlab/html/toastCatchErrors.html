toastCatchErrors

Catch errors generated by the toast libraries.

Syntax

toastCatchErrors()

Notes

Overrides the default error handling method of the toast libraries. By default, an error detected by a library function terminates the calling process. For the mex files in the TOAST toolbox this is generally undesirable, because it kills the Matlab process. Using toastCatchErrors before calling any other toast functions modifies the error handling strategy to output an error message and return to the Matlab prompt.

Note that this has an effect only on errors that are trapped by the libraries. Any unchecked errors can still crash the Matlab session.
