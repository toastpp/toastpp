toastMeshData

Return mesh geometry data: Node coordinate list, element connectivity list and element type list.

Syntax

[vtx,idx,eltp] = toastMeshData(hmesh)

Parameters

[In] hmeshhandlemesh handle
[Out] vtxreal vector n x dmatrix of node coordinates
idxindex matrix m x smatrix of node indices for each element
eltpelement type list m x 1list of element type indices

Notes

Returns the node coordinates and element vertex index list for a mesh.

The returned vertex list is a real matrix of dimension n x d, where n is the number of nodes in the mesh, and d is the mesh dimension.

The returned index list is an integer matrix of dimension m x s, where m is the number of mesh elements, and s is the maximum number of nodes per element. The index list is 1-based. For elements containing fewer than s nodes, the unused entries are set to 0.

For a list of element type indices, see documentation of function toastMakeMesh.
