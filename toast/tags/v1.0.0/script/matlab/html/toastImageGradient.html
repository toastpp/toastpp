toastImageGradient

Return the gradient of an image.

Syntax

grad = toastImageGradient(hBasis,img)

Parameters

[In] hBasishandlebasis mapper handle
imgreal or complex vector [g]image
[Out] gradreal or complex matrix [n x g]image gradient

Notes

Calculates the gradient of the image given in fine resolution (g-basis). Any pixels outside the object domain are removed from the calculation, to suppress gradients at the domain boundary. The gradients are returned as a dense matrix of dimension n x g, where n is the domain dimension (2 or 3). Real and complex images are supported.
