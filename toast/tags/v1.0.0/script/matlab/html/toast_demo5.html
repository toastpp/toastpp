TOAST Demo 5: Structural priors

The reconstruction can be improved by introducing prior knowledge about the expected structure of the images.

In this example, the prior is used to scale down the regularisation parameter along known edges in the images. This prevents the edges to be smoothed out by the prior.

Select one of the prior references. Adjust the smoothing factor (smaller values make the edge areas thinner and increase background noise), the threshold value (larger values decrease the magnitude of the edge scaling) and hyperparameter (balancing the relative weighting between prior and likelihood terms).

Then click 'Run' to start the reconstruction.
