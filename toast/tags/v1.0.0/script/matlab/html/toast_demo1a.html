TOAST Demo: Time-dependent forward solver

This demo shows an example for applying the TOAST forward solver in a time-dependent problem.

It generates the transilluminated temporal intensity profile, or temporal point spread function (TPSF) at the detector positions, arising from a delta-source in time.

TOAST uses a finite-difference scheme to propagate the solution in time. You can select the time window, number of temporal samples, and the coupling factor connecting the time steps.

A coupling factor of 0.5 corrresponds to the Crank- Nicholson scheme. A factor of 1 corresponds to an implicit scheme.

This demo allows to calculate the forward solution either in a homogeneous ('flat') or inhomogeneous ('blobs') medium. It can be seen that even in an inhomogeneous case, the time profile is smooth without obvious features, demonstrating the low TPSF information content.

For the reconstruction, usually a small number of features is extracted such as integrated intensity, mean arrival time, or the Fourier-transformed signal sampled at a small number of frequencies.
