toastFwdCW

The TOAST forward solver for diffusion light transport (continuous wave version: real-valued problem).

Syntax

cwdata = toastFwdCW (prm)

Parameters

[In] prmstructcontains options and parameters for the forward solver
[Out] cwdatareal [nqm]log intensity boundary data for all source and detector combinations.
Dimensions: nqm = number of measurements

Notes

This is a high-level function that contains the TOAST FEM continuous-wave forward solver for modelling steady-state diffuse light transport in a scattering medium. It calculates the boundary measurements for a given FEM mesh, absorption, scattering and refractive index distributions, and boundary source and detector profiles.

All model parameters and options are provided by the 'prm' input argument. For a description of the prm structure, see Parameter structure.

prm can either be generated manually, read from a file with toastReadParam, or defined interactively with a GUI tool, fwd_gui.

The calculated boundary data are returned in cwdata, as log intensity (boundary exitance integrated over each detector profile). The data vector is a 1-D real vector, containing the measurements for all source and detector combinations defined in the QM link list. The data are arranged so that the detector index changes most rapidly.

If the prm.lnampfile field is defined, then the boundary data are also written to the files specified by this entry.
