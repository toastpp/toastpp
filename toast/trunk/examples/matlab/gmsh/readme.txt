Examples for gmsh support in Toast. The mesh files in this directory are in gmsh format. They can be read into toast with the gmsh filter.

gmsh is a mesh generator for unstructured FEM grids. It can be downloaded from

http://geuz.org/gmsh/

The almi5.msh and almi5-lowres.msh files have been included from the SimNIBS example package with kind permission by Alex Thielscher. SimNIBS is available at

http://simnibs.de/
