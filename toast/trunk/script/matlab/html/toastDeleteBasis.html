toastDeleteBasis

De-allocate a basis mapper object from dynamic heap.

Syntax

toastDeleteBasis(hBasis)

Parameters

[In] hBasishandlebasis mapper handle

Notes

Deletes a basis mapper from dynamic heap. hBasis is a handle to the mapper obtained by a previous call to toastSetBasis. The handle is no longer valid after this call.
