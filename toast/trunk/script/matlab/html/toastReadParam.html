toastReadParam

Read configuration parameters and options for the forward or inverse solver from a TOAST parameter file into a MATLAB data structure.

Syntax

prm = toastReadParam(prmfile)

Parameters

[In] prmfilestringparameter file name
[Out] prmstructstructure containing the parsed parameters

Notes

This function reads the toast forward solver parameters used by toastFwd or the reconstruction parameters used by toastRecon from a file and stores them in a structure that can be passed directly to toastFwd or toastRecon.
