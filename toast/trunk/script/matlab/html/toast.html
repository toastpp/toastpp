Toast++ is an open-source software suite for light transport modelling and image reconstruction in diffuse optical tomography. It uses the finite element method to model diffuse light propagation in inhomogeneous, highly scattering media. 3D image reconstruction is performed by a nonlinear approach that fits the diffusion model to the data by iteratively updating the model parameters. A regularisation term can be added to the objective function to incorporate prior knowledge.
