toastReadQM

Read a QM (source-detector) specification

Syntax

toastReadQM(hMesh,qmfile)

Parameters

[In] hMeshhandlemesh handle
qmfilestringQM file name

Notes

Reads a QM (source-detector) specification from a file into a TOAST mesh. The QM file contains information of source and detector positions, as well as a 'link list' defining the associations between sources and detectors.

See also

toastSetQM, toastGetQM, toastWriteQM
