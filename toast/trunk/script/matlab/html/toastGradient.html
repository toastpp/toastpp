toastGradient

Return gradient of objective function, dψ / dx.

Syntax

grad = toastGradient (hMesh, hBasis, qvec, mvec, mua, mus, ref, freq, data, sd, solver, tol)

Parameters

[In] hMeshhandlemesh handle
hBasishandlebasis mapper handle
qveccomplex sparse matrix [h x nq]array of source vectors
mveccomplex sparse matrix [h x nm]array of measurement vectors
muareal vector [h]nodal absorption coefficients
musreal vector [h]nodal scattering coefficients
refreal vector [h]nodal refractive index values
freqrealsource modulation frequency [MHz]
datareal vector [2nqm]measurement data vector
sdreal vector [2nqm]vector of measurement standard deviations
solverstringlinear solver method [DIRECT | CG | BICG | BICGSTAB | GMRES | GAUSSSEIDEL]
tolrealsolver tolerance (ignored for solver DIRECT)
[Out] gradreal vector [2s]gradient of objective function

Dimensions: h = size of h-basis, s = size of s-basis, nq = number of sources, nm = number of detectors, nqm = number of measurements

Notes

This function runs the forward solver with the nodal optical parameters provided, and calculates the derivative of the objective function ψ with respect to the absorption and diffusion parameters x = [μ ; κ].

The returned gradient vector dψ/dx is expressed in the inverse solution basis (s-basis). It contains the gradients with respect to μ and κ in two consecutive blocks: grad = [dψ/dμ ; d&psi/dκ]
